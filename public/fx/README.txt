Custom FX samples — drop real recordings here and they automatically
replace the built-in synthesized effects (no code changes needed).

File names (mp3):
  air_horn.mp3   — the HORN button
  riser.mp3      — the RISER button
  drop.mp3       — the DROP button
  laser.mp3      — the LASER button
  brake.mp3      — the BRAKE spin-down sound (the slowdown+blend still happens)

Tips:
- Keep them short (1–4s) and already loud/mastered; they play at ~90%
  of the FX volume with a touch of the app's reverb.
- Use royalty-free sources (e.g. CC0 packs) since this repo is public.
- After adding files, commit + push — the deploy picks them up.
